# Визуальная система

## Типографика
- Заголовки: **Prata** (serif, акцентная, «дорогая»)
- Текст: **Sora** (sans, читабельная)
- Моно/код: **JetBrains Mono** (для визуалов с кодом)

## Цвета (токены)
- `--bg-0`: #0b1110
- `--bg-1`: #0f1e1b
- `--bg-2`: #13312c
- `--text-1`: #f4f1ea
- `--text-2`: #c8d6d0
- `--muted`: #8ba39a
- `--accent-1`: #ffb347 (amber)
- `--accent-2`: #63f7d3 (mint)
- `--accent-3`: #ff6b4a (coral)
- `--stroke`: rgba(255,255,255,0.12)
- `--glass`: rgba(255,255,255,0.06)

## Градиенты
- Фон: radial(#1b3a31 → transparent) + linear(#0b1110 → #13312c)
- Акцент: linear(120deg, #63f7d3 0%, #ffb347 60%, #ff6b4a 100%)

## Пространство и сетка
- Базовая сетка: 8px
- Контейнер: 1200–1240px
- Радиусы: 12px (sm), 20px (md), 28px (lg)

## Эффекты
- Glass: фон `rgba(255,255,255,0.06)` + blur 16–24px + тонкий stroke
- Glow: мягкая подсветка `rgba(99,247,211,0.35)`
- Тени: 0 20px 60px rgba(0,0,0,0.35)

## Компоненты
- **Primary CTA**: градиент + мягкий glow
- **Secondary CTA**: прозрачная с stroke
- **Карточки**: glass + hover‑lift + glow
- **FAQ**: аккордеон с плавной анимацией
- **Теги**: pill‑бейджи с контуром

## Motion
- Появление секций: 500–700ms, ease‑out
- Hover: 200–300ms
- Scroll‑reveal: стаггер 80–120ms
